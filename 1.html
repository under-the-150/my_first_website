HTML이란 무엇인가?

Hypertext Marup Language(HTML) is the standard markup language for creating web pages and web appliciations. Web browseres receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, imges and other objects, such as interactive forms, may be embedded into the rendered page.
